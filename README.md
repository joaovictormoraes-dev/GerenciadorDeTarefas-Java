# 🚀 Gerenciador de Tarefas em Java

Este é um projeto-piloto feito por mim desenvolvido para praticar os fundamentos da linguagem Java e Programação Orientada a Objetos (POO).

## 🛠️ Funcionalidades
- **Adicionar Tarefas:** Registo de novas atividades.
- **Listar Tarefas:** Visualização com status [OK] para concluídas e [ ] para pendentes.
- **Remover Tarefas:** Exclusão de itens da lista.

## 🧠 Conceitos Praticados
- **POO:** Classes, atributos e métodos.
- **Coleções:** Uso de `ArrayList` para armazenamento dinâmico.
- **Estruturas:** `switch case`, `do-while` e tratamento de erros com `try-catch`.
- **Git/GitHub.** Fluxo de versionamento e organização de repositório.
